# Contributing to ProjectBeast

## プロジェクトに参加する方法

ProjectBeastへの貢献に興味を持っていただきありがとうございます！

## 最新情報を受け取る

- **Watch**: GitHubの[Watch](https://github.com/TakebayashiNaoya/ProjectBeast/subscription)機能を使用して、プロジェクトの更新通知を受け取ることができます
- **Star**: リポジトリに[Star](https://github.com/TakebayashiNaoya/ProjectBeast)をつけて、後で簡単に見つけられるようにします
- **Fork**: 自分のコピーを作成して、独自の変更を加えることができます

## 開発環境

このプロジェクトは以下の環境で開発されています：

- Windows環境
- Visual Studio
- k2Engine / k2EngineLow
- DirectX 12

## プルリクエスト

改善案やバグ修正がある場合は、プルリクエストを送信してください。

1. リポジトリをForkする
2. 新しいブランチを作成する
3. 変更を加える
4. プルリクエストを作成する

## 問題の報告

バグや問題を見つけた場合は、GitHubのIssuesで報告してください。
